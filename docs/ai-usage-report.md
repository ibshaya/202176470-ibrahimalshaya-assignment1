# AI Usage Report
## Tools Used & Use Cases

ChatGPT
Used primarily as a consultant and guide for structuring the project folder, organizing files (HTML, CSS, JS), and planning the layout of sections in the portfolio website. ChatGPT helped clarify best practices for responsive design and suggested interactivity features.

Claude
Used for AI-assisted code generation, including generating sample HTML sections, CSS styling snippets, and JavaScript functionalities such as smooth scrolling and mobile menu toggling. Claude provided me with a code that accelerated development while ensuring modular, readable structure.

## Benefits & Challenges

Benefits:

Accelerated development by providing structured examples and reusable code snippets.

Helped clarify best practices for responsive web design and UI organization.

Assisted in debugging and validating JavaScript logic, saving time on trial-and-error.

## Challenges:

Some generated code required manual review to ensure it adhered to the project requirements and personal coding style.

Some adjustments were needed to maintain originality and correct AI suggestions for context-specific functionality.

## Learning Outcomes


Learned how to integrate interactive JavaScript features, including theme toggles and dynamic greetings.

Gained experience in reviewing and modifying AI-generated code to fit specific project needs while maintaining clarity and correctness.

Learned effective ways to combine multiple AI tools in a workflow to streamline coding tasks without over-relying on them.

## Responsible Use & Modifications

All AI-generated suggestions were carefully reviewed, tested, and modified to ensure they met assignment requirements.

Code was adjusted for correctness, readability, and originality, avoiding direct copy-paste where necessary.
